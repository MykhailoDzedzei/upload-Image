# AngularJS-Todos
To start project you need to have already running MongoDB and then do those steps:
- npm install
- gulp
- node app.js(inside src/server folder)

###Technologies:

##### Actual:

- Node / npm
- Gulp / Babel / WebPack
- Bootstrap

- MongoDB
- Express
- gm
- fs
